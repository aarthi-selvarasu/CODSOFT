# Task 1: Titanic Survival Prediction

## Description
This project predicts whether a passenger survived the Titanic disaster using machine learning techniques.

## Dataset
- Titanic Dataset (Kaggle)
- File used: train.csv

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Steps Performed
- Data cleaning and preprocessing
- Feature encoding
- Model training using Decision Tree
- Prediction and evaluation

## Output
The model predicts whether a passenger survived or not and displays survival details from the dataset.
